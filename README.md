# 🎓 SUAI Queue Bot (Помощник по очередям ГУАП)

![Go](https://img.shields.io/badge/Go-00ADD8?style=for-the-badge&logo=go&logoColor=white)
![Telegram](https://img.shields.io/badge/Telegram-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)

**SUAI Queue Bot** — это Telegram-бот, созданный специально для студентов СПбГУАП. Он помогает навсегда забыть про споры у дверей кафедр и автоматизирует процесс очереди на сдачу лабораторных, курсовых работ и зачетов. 

Вставай в список дистанционно, следи за своим местом и не трать время на ожидание в коридоре! 🚀

---

## ✨ Возможности

- 📝 **Регистрация студентов**: Простая регистрация по имени с привязкой к Telegram ID.
- ➕ **Управление очередью**: Возможность встать в очередь, посмотреть текущий список и выйти из нее.
- 🧹 **Умная автоочистка**: Бот автоматически удаляет из очереди студентов, которые находятся в ней слишком долго (более 25 минут), и присылает им уведомление. Это защищает очередь от "мертвых душ".
- ⚙️ **Настройки профиля**: Возможность в любой момент изменить свое имя, которое отображается в очереди.

## 🛠 Технологии

- **Язык:** [Go (Golang)](https://go.dev/)
- **Библиотека для Telegram API:** [Telebot v3](https://gopkg.in/telebot.v3)
- **Конфигурация:** [godotenv](https://github.com/joho/godotenv)
- **Хранение данных:** В оперативной памяти (In-Memory).

## 🚀 Установка и запуск

### Требования
- Установленный **Go** (версия 1.18 или выше).
- Токен Telegram бота (получить можно у [@BotFather](https://t.me/BotFather)).

### Инструкция

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/kandreyss/suai-queue.git
   cd suai-queue
   ```

2. **Установите зависимости:**
   ```bash
   go mod download
   ```

3. **Настройте переменные окружения:**
   Создайте файл `.env` в корневой директории проекта и добавьте в него токен вашего бота:
   ```env
   TOKEN=your_telegram_bot_token_here
   ```

4. **Запустите бота:**
   ```bash
   go run cmd/bot/main.go
   ```

После успешного запуска в консоли появится сообщение: `Бот запущен 🚀`

## 📱 Как использовать бота

1. Найдите бота в Telegram и нажмите **"Старт"** (или отправьте `/start`).
2. Пройдите регистрацию, отправив команду `/register` и введя свое настоящее имя.
3. Используйте удобное кнопочное меню:
   - **➕ Встать в очередь** — занять место в конце списка.
   - **📋 Посмотреть очередь** — вывести текущий пронумерованный список студентов.
   - **➖ Выйти из очереди** — освободить место (не забудьте нажать после сдачи работы!).
4. Для смены имени используйте команду `/settings`.

## 📂 Структура проекта

```text
suai-queue/
├── cmd/bot/
│   └── main.go              # Точка входа, инициализация бота и планировщика задач
├── internal/
│   ├── config/              # Загрузка конфигурации из .env
│   ├── handlers/            # Обработчики команд Telegram (кнопки, текст, меню)
│   │   └── sessions/        # Хранилище состояний пользователей (FSM) для диалогов
│   └── service/             # Бизнес-логика работы со студентами (In-Memory БД)
├── pkg/
│   ├── queue/               # Логика работы с очередью и фоновая очистка (Cleanup)
│   └── student/             # Доменная модель студента
├── .env                     # Файл с переменными окружения (не в коммите)
├── go.mod                   # Зависимости проекта
└── README.md                # Описание проекта
```
